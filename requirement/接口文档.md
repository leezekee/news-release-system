# 接口文档

>响应统一结构:
>
>```json
>{
>    "code": "返回的状态码",
>    "msg": "返回的信息",
>    "data": "返回的数据"
>}
>```

## 主编

### /chiefEditor/login

* 方法：`POST`

* 参数：

  | 参数名   | 类型   | 说明   |
  | -------- | ------ | ------ |
  | username | String | 用户名 |
  | password | String | 密码   |

* 返回值：

    | 参数名 | 类型   | 说明 |
    | ------ | ------ | ---- |
    | token  | String | 令牌 |

* 返回值：

   | 参数名 | 类型   | 说明 |

   | token  | String | 令牌 |

### /chiefEditor/info

* 方法：`PUT`

* 参数：

  | 参数名                 | 类型    | 说明   |
  |---------------------| ------- |------|
  | id                  | Integer | 主编号  |
  | name（可选）            | String  | 用户别名 |
  | age（可选）             | Integer | 年龄   |
  | gender（可选）          | String  | 性别   |
  | address（可选）         | String  | 地址   |
  | idCardNumber（可选）    | String  | 证件号  |
  | telephoneNumber（可选） | String  | 电话   |
  | email（可选）           | String  | 邮箱   |
  | introduction（可选）    | String  | 介绍   |

* 返回值：

  无

## 新闻

  | time    | DateTime | 新闻编辑时间 |
  | imageId | Integer  | 图片id       |

* 返回值：

  无

### /news

* 参数：

  | title   | String   | 新闻标题     |

  | imageId | Integer  | 图片id       |

  无

### /news

* 参数：

  | ------ | ------- | ------ |
  | id     | Integer | 新闻id |
  
 * 返回值：

   无

### /news/{id}

   * 方法：`GET`

   * 参数：

     | 参数名 | 类型    | 说明   |

### /news/review

   * 方法：`PUT`

   * 参数：

     | 参数名                | 类型    | 说明                                                   |
     | --------------------- | ------- | ------------------------------------------------------ |
     | id                    | Integer | 新闻id                                                 |
     | status（可选）        | Integer | 新闻是否发布（1为审核通过，-1为审核不通过，0为审核中） |
     | reviewComment（可选） | String  | 审核意见（审核通过无需）                               |
     
   * 返回值：

     无
